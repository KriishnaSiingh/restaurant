

## Lounge — Premium Restaurant + Hotel Booking Website

A dark luxury website inspired by the Lounge template, featuring a black + gold color scheme, elegant serif typography, smooth animations, and a premium 5-star brand feel.

### Design System
- **Theme**: Dark background (#0A0A0A) with gold accents (#C9A96E), warm ivory text
- **Typography**: Playfair Display (headings) + Inter (body) — luxury editorial feel
- **Effects**: Glassmorphism cards, fade-in scroll animations, parallax hero, hover micro-interactions
- **Fully responsive**, mobile-first

---

### Pages & Features

**1. Home Page**
- Full-screen hero with image background, animated headline, and dual CTA (Restaurant / Hotel)
- "Today's Offers" banner with discount badges
- Featured dishes carousel
- Featured rooms preview
- "Dinner + Stay" combo package section with discount badge
- Testimonials slider
- Quick booking form (restaurant or hotel toggle)
- Floating "Book Now" button

**2. Restaurant Page**
- Menu with category filters (Veg, Non-Veg, Drinks, Desserts)
- Interactive menu cards with hover effects, "Add to Cart" and "Wishlist" buttons
- Chef's Specials section
- Table reservation form (date, time, guests) with simulated confirmation

**3. Hotel Page**
- Room listings (Standard, Deluxe, Suite) with images, pricing, amenities
- Room detail modal with image gallery
- Availability calendar UI
- Hotel booking form (check-in, check-out, rooms, guests) with simulated confirmation

**4. About Page**
- Brand story with parallax sections
- Chef introduction with photo
- Restaurant ambience showcase

**5. Gallery Page**
- Masonry/grid layout with category tabs (Food, Rooms, Interior)
- Lightbox zoom preview

**6. Contact Page**
- Contact form with validation
- Map placeholder UI
- Social media links

**7. Login/Signup Page**
- Frontend-only auth UI with tabs

---

### Advanced Features
- **Sticky navbar** with background blur on scroll
- **Smart search bar** — filter dishes and rooms by keyword
- **Wishlist** — save dishes/rooms to localStorage
- **Cart UI** — add food items, view cart drawer
- **Reviews carousel** with star ratings
- **Loading animation** on initial page load
- **Lazy loading** for images
- **Smooth scroll animations** throughout (fade-in on scroll via Intersection Observer)
